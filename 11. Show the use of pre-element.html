Using the <pre> Element

Text in the <pre> element is displayed in a fixed-width font and respects whitespace and line breaks.

    This is an example of preformatted text.
    Spaces and line breaks
        are preserved.

    You can also use it to display code snippets:
    for (int i = 0; i < 10; i++) {
        console.log("Hello World");
    }
