HTML Character Entities

Here are some examples of HTML character entities:

  • < : Less-than sign (<)
  • > : Greater-than sign (>)
  • & : Ampersand (&)
  • " : Double quotation mark (")
  • ' : Single quotation mark (')
  • © : Copyright symbol (©)
  • ® : Registered trademark (®)
  • € : Euro currency (€)
  • £ : British pound (£)

Using these entities allows us to include special characters in our HTML without confusing the browser.
